# 项目列表页面

## 新增功能

创建了一个独立的项目列表页面 `/lovart/projects`，专门用于展示当前用户的所有项目。

## 页面路径

- **主页**: `/lovart` - 欢迎页面，包含搜索和快速访问
- **项目列表**: `/lovart/projects` - 显示用户所有项目的专用页面
- **画布编辑器**: `/lovart/canvas` - 项目编辑页面

## 功能特性

### 1. 项目展示
- 网格布局显示所有项目
- 响应式设计：1/2/3/4 列（根据屏幕大小）
- 显示项目缩略图、标题和最后编辑时间
- 点击项目卡片进入编辑页面

### 2. 搜索功能
- 实时搜索项目标题
- 搜索结果即时更新
- 未找到结果时显示提示信息

### 3. 新建项目
- 右上角"新建项目"按钮
- 点击跳转到画布编辑器
- 空状态时显示"创建项目"引导

### 4. 用户状态
- **未登录**: 显示登录引导页面
- **加载中**: 显示加载提示
- **无项目**: 显示空状态和创建引导
- **有项目**: 显示项目网格

### 5. 侧边栏导航
- **Home 图标**: 返回主页 (`/lovart`)
- **Folder 图标**: 项目列表 (`/lovart/projects`) - 当前页面时高亮
- **User 图标**: 用户相关功能（待实现）

## 视觉设计

- 简洁的白色背景
- 清晰的标题和搜索栏
- 项目卡片悬停效果
- 激活状态的侧边栏图标（灰色背景）
- 平滑的过渡动画

## 技术实现

### 文件结构
```
src/
├── app/
│   └── lovart/
│       ├── page.tsx              # 主页
│       ├── projects/
│       │   └── page.tsx          # 项目列表页面（新增）
│       └── canvas/
│           └── page.tsx          # 画布编辑器
└── components/
    └── lovart/
        ├── DashboardSidebar.tsx  # 侧边栏（已更新）
        └── ProjectCard.tsx       # 项目卡片组件
```

### 数据加载
- 使用 `useSupabase` hook 连接数据库
- 从 `projects` 表加载用户项目
- 按 `updated_at` 降序排序
- 实时搜索过滤

### 路由导航
- 使用 Next.js Link 组件
- 使用 `usePathname` 检测当前路径
- 动态应用激活状态样式

## 用户体验

1. **访问项目列表**
   - 点击侧边栏的 Folder 图标
   - 或直接访问 `/lovart/projects`

2. **搜索项目**
   - 在搜索框输入关键词
   - 实时过滤项目列表

3. **创建新项目**
   - 点击"新建项目"按钮
   - 或在空状态时点击"创建项目"

4. **打开项目**
   - 点击任意项目卡片
   - 跳转到画布编辑器
