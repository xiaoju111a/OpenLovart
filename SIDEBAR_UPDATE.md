# 主页侧边栏项目按钮更新

## 更新内容

在主页（`/lovart`）的侧边栏中，将"项目"按钮（Folder 图标）改为可点击的链接。

## 功能说明

1. **项目按钮** - 文件夹图标，点击跳转到 `/lovart` 页面（项目列表）
2. **激活状态** - 当前在项目页面时，按钮显示为激活状态（灰色背景）
3. **Logo 可点击** - 顶部的 Lovart Logo 也可以点击返回项目列表

## 视觉效果

- **未激活状态**: 灰色图标，悬停时背景变灰，图标变黑
- **激活状态**: 灰色背景，黑色图标
- **Logo**: 黑色圆形背景，白色 "L" 字母，悬停时背景变深灰

## 技术实现

- 使用 `usePathname()` hook 检测当前路径
- 根据路径动态应用激活样式
- 移除了原来的 Home 按钮，因为项目列表就是主页

## 文件修改

- `src/components/lovart/DashboardSidebar.tsx` - 更新侧边栏组件
  - 添加 `'use client'` 指令
  - 使用 `usePathname` 检测当前路径
  - 将 Folder 按钮改为 Link 组件
  - 添加激活状态样式
  - Logo 改为可点击的链接

## 用户体验

用户现在可以：
1. 点击侧边栏的文件夹图标查看所有项目
2. 在项目页面时，文件夹图标会高亮显示
3. 点击 Logo 快速返回项目列表
